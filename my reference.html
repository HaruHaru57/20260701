My Reference

HTML

  • <a href="url">〇〇</a>　　URLに飛ぶリンクを貼れる
  • <h1,2,3,4...> 見出し
  • <p> 段落
  • <ul> 黒丸の箇条書き
  • <ol> 数字の箇条書き
  • <li> 箇条書きの中身
  • <button onclick="alert('〇〇')">ボタン</button>　　ボタンを押すとアラートが出る onclick内がボタンを押したときの処理
  • <input type="text" placeholder="〇〇">　　入力欄を作れる
  • <div>　　divタグで囲むとまとめて装飾できる
  • <br>　　改行 </br>はいらない
  • <img src="url" alt="〇〇" width="〇〇px">　　画像を表示できる

CSS

  • <style>　　CSSを使える
  • color:　　文字色を変える
  • font-size:　　文字サイズを変える
  • background-color:　　背景色を変える
  • padding:　　余白を変える
  • border:　　枠線を変える
  • border-radius:　　角を丸くする
  • cursor:　　カーソルの形を変える

JavaScript

  • <script>　　JavaScriptを使える
  • function 〇〇() { }　　関数を作れる
  • alert('〇〇')　　アラートを出せる
  • Math.floor(Math.random() * 〇〇)　　ランダムな数字を出せる
    Math.random()は0から1の間のランダムな小数を返す
    Math.floor()は()内の値の小数点以下を切り捨てる
  • onclick="〇〇()"　　ボタンを押したときに関数を呼び出せる
  • () : 関数を呼び出す
  • [] : 配列のn番目の要素を取得する
  • document.getElementById("〇〇").innerHTML = "△△"　　idが〇〇の要素の中身を△△に変える
  • document.getElementById("〇〇").textContent = "△△"　　idが〇〇の要素のテキスト内容をそのまま△△に変える
  • document.getElementById("〇〇").value = "△△"　　idが〇〇の入力欄の値を△△に変える
  • document.getElementById("〇〇").style.backgroundColor = "△△"　　idが〇〇の要素の背景色を△△に変える
    ※style.〇〇 = "△△"で〇〇の部分を変えると他の装飾も変えられる
    例
    • .style.color 文字の色
    • .style.fontSize 文字サイズ
    • .style.backgroundColor 背景色
    • .style.fontWeight 文字の太さ
    • .style.border 枠線
    • .style.width 幅
    • .style.padding 余白
    • .style.display 表示/非表示
    • 条件A && 条件B 条件Aと条件Bの両方が真の場合に真になる (かつ)
    • 条件A || 条件B 条件Aか条件Bのどちらかが真の場合に真になる (または)
    • document.getElementById("〇〇").addEventListener("□□", △△)　　idが〇〇の要素を□□したときに△△の処理をする
    • console.log("〇〇")　　画面の見た目を変えずに、コンソールに〇〇を表示する
